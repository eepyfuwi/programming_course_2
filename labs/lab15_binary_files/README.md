### ЗАДАНИЯ

#### ЦВЕТОВОЙ ФИЛЬТР. Имеется изображение в формате .bmp. Напишите программу, с помощью которой можно менять интенсивность красной, зеленой или синей составляющих цветов пикселей в указанных прямоугольных частях рисунка.

#### ШИФРОВАНИЕ.  В файле имеется текст, который нужно зашифровать. Для шифрования предлагается метод, который назовем методом шифрования с помощью бинарного сдвига на заданное количество двоичных разрядов.

Суть метода бинарного сдвига.

#### Текст последовательно разбивается на части одинаковой длины, последняя часть может оказаться короче. Внутри каждой части  текста, рассматриваемой как последовательность бит, выполняется циклический сдвиг вправо или влево на заданное количество двоичных разрядов. Положительное значение сдвига, т.е. положительное количество двоичных разрядов, задает сдвиг вправо, отрицательное  – сдвиг влево. Если последняя часть текста короче остальных, то она перед сдвигом дополняется до нужной длины пробелами справа. Преобразованные части текста одна за другой записываются в выходной файл.

#### Написать функции шифрования и дешифрования текста методом бинарного сдвига.

Входные данные:

текстовый файл;

RecordLength – длина шифруемой части текста;

Shift – размер сдвига.

Результат: зашифрованный / дешифрованный текст в новых файлах.

## Формат ввода/вывода

**Ввод:**
- «Цветовой фильтр»: исходный файл изображения `.bmp` и параметры фильтра (прямоугольники и изменение интенсивности каналов), как указано в задании.
- «Шифрование»: текстовый файл, `RecordLength` (длина блока) и `Shift` (величина циклического сдвига).

**Вывод:**
- «Цветовой фильтр»: обработанное изображение в новом файле `.bmp`.
- «Шифрование»: зашифрованный/дешифрованный текст в новых файлах.
